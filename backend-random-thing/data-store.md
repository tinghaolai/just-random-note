## Float on money

Never use float/double to store money, 
or anything that needs to be precise.

Use 'string' type instead, such as 'decimal(10,2)'.

## 浮動貨幣

永遠不要使用浮點數/雙精度來存儲金錢，或任何需要精確的東西。

相反，使用 'string' 類型，例如 'decimal(10,2)'。
